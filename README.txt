================================================================================
  TaskManagementWithFrameWork
  タスク管理 + 日記 + フレームワーク活用 Webアプリ
================================================================================

■ 概要
  複数のタスク管理フレームワークを統合したWebアプリケーションです。
  タスク管理、日記、ポモドーロタイマーなどの機能を1つの画面で利用できます。

■ 主な機能

  【タスク管理】
  - タスクの作成・編集・削除・完了
  - タスクの階層化（親子関係）
  - ドラッグ&ドロップ / 矢印ボタンによる並べ替え・階層移動
  - 3つの表示モード（詳細 / コンパクト / グリッド）
  - カードの色でリスク可視化（赤=リスク / 青=進行中 / 緑=完了）
  - 着手・未着手バッジ
  - 繰り返しタスク（日次/週次/月次/年次/カスタム）
  - 完了タスクの自動アーカイブ
  - 削除タスクの復元（過去100件）

  【タスクセット】
  - よく使うタスクの組み合わせをテンプレートとして保存
  - タスクセットの公開・インポート
  - いいね機能・ワード検索

  【ポモドーロタイマー】
  - 25分作業 + 5分休憩のサイクル
  - 全画面表示でタスク情報を確認しながら集中
  - 終了時にアラーム音で通知
  - 作業時間を自動で実績に加算

  【アイゼンハワーマトリクス】
  - 緊急性 x 重要性の2軸で未完了タスクを配置
  - 自由にドラッグして整理
  - 配置図の保存・読み込み

  【パレート分析】
  - 作業時間の上位20%を可視化
  - タスクごとの時間配分を棒グラフで表示

  【日記】
  - 日付ごとの日記作成（同日は追記）
  - 日記の公開・非公開設定
  - みんなの日記（他ユーザーの公開日記を閲覧）
  - いいね・リプライ機能
  - 期間絞り込み・ワード検索

  【作業記録】
  - タスクごとの作業ログを日付付きで蓄積
  - 作業ログ・タスク作成・完了・削除の一覧表示
  - 統計モード（日ごとの作業数・作業時間）
  - フィルター機能（種別・期間）
  - ページネーション（20件ずつ表示）

  【執事アバター】
  - 画面右下に執事アイコンが表示
  - タスク状況に応じた励ましメッセージを表示
  - アイコン画像・吹き出し文字数・指示内容をカスタマイズ可能
  - ドラッグで移動可能・非表示にも対応

  【ウェルカムページ】
  - 初回アクセス時に豆知識やライフハックをランダム表示
  - 2秒でフェードアウトしてタスクページに遷移

  【設定】
  - ダークモード
  - フォントサイズ・フォント種類の変更
  - 執事アバターのカスタマイズ

  【その他】
  - ユーザー登録・ログイン・パスワード変更・リセット
  - マイページ（プロフィール編集・アイコン設定）
  - 管理者ページ（バグ報告管理・公開コンテンツ管理）
  - バグ報告機能
  - ヘルプページ（機能説明 + ハンズオンチュートリアル）
  - スマートフォン対応（レスポンシブ）

■ 技術スタック

  フロントエンド: Next.js (App Router) + React + TypeScript
  バックエンド:   Next.js API Routes
  データベース:   Turso (libsql) - SQLite互換クラウドDB
  スタイリング:   CSS Modules
  ホスティング:   Vercel

■ ローカル開発

  1. 依存パッケージのインストール
     npm install

  2. 環境変数の設定（.env.local を作成）
     TURSO_DATABASE_URL=libsql://your-db.turso.io
     TURSO_AUTH_TOKEN=your-token
     GEMINI_API_KEY=your-key（任意）

  3. 開発サーバーの起動
     npm run dev

  4. ブラウザでアクセス
     http://localhost:3000

  ※ TURSO_DATABASE_URLを省略するとローカルファイルDB（data/todos.db）を使用
